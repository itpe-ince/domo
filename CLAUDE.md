## Project Overview

**Domo** is a social network web app for emerging/beginner artists and art students worldwide. It combines SNS, blogging, artwork showcase (photo/video uploads), patronage (sponsorship), and artwork auctions.

Key concepts:
- "Blue Bird" sponsorship system — low-barrier micro-patronage for artists
- Global artist index — tracking and ranking emerging artists by transaction activity
- Growth hacking funnel — build user base first, then convert to buyers/collectors
- Target audience: emerging artists in developing regions (Southeast Asia, Latin America, Eastern Europe) who lack gallery access

## Current State

This project is in the ideation/brainstorming phase. There is no codebase yet — only a README.md with business strategy notes and a `docs/` directory with bkit PDCA status files. The development pipeline level is set to Dynamic (fullstack with BaaS).
